## ✋🏻简介
这是一个基于`React` `Ffmpeg` `Electron`的**直播录制软件**。支持监控直播，可以帮助用户简单便捷的对直播进行录制并保存为MP4格式的视频，

#### 已支持平台
- [x] youtube
- [x] twitch
- [x] tiktok
- [x] 抖音
- [x] 快手
- [x] B站
- [x] 网易cc
- [x] 花椒
- [x] weibo
- [ ] 虎牙
- [ ] 斗鱼

#### ❓如何在本地运行？
```shell
pnpm install
```
```shell
pnpm debug
```
